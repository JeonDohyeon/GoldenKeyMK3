1. 리듬마블 로고 + 타이머 + 바퀴수 + 진행방향 컨트롤러
2. 씬 변경 버튼 옮기기
3. !픽 로깅 시스템 구축
4. 재추첨 기능 추가
5. 도움말 공개 기능
6. 창모드 변경
7. 딜레이 추가

구체적인 계획

1. 컨트롤러 추가
    - 위치는 씬 변경 버튼 근처로 하기
    - 타이머 시작/일시정지/+/- 1분 기능
    - 바퀴수 +/- 1바퀴 기능
    - 진행방향 변경 기능
2. 씬 변경 버튼 옮기기
    - 좌측 상단이 좋아보임
    - 곡 추첨 씬 좌측 리스트 높이를 조금 줄이면 될 듯
    - 어차피 속도는 1/프레임이니까 적당함
3. !픽 로깅 시스템
    - 좌측 리스트 아래에 최근 5개만 띄우면 될듯
    - 시간이 지나면 사라지게 할 예정
4. 재추첨 기능 추가
    - 추첨 버튼 바로 위에 올려둘 예정
    - 현재 상태가 idle, active, result인데 result에만 띄우고
      클릭하면 그 표만 지우고 다시 뽑는 걸로
5. 도움말 공개 기능
    - 뱅하싶 칸에 넣을 예정 (가장 쉬움)
6. 창모드 변경 (v)
    - 창모드 변경 자체는 쉬움
    - 나머지 씬들에 들어간 getscreenwidth랑 getscreenheight를
      상수 하드코딩으로 갈아끼우는게 귀찮아서 그렇지
7. 딜레이 추가
    - 이건 방장이랑 한번 더 상의를 해야겠음
    - 방장은 표 추가시 1분 딜레이가 좋겠다고 했지만
      어차피 1곡 플레이 하고 나면 대략 4~5분이라 애매함
